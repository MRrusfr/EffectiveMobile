- [OpenAPI-спецификация (YAML)](./task3_api/task3_api.yaml) — можно открыть в [Swagger Editor](https://editor.swagger.io/)
- [Алгоритм обработки запроса (draw.io)](./task3_api/task3_algorithm.drawio)

<details>
<summary>Краткое описание эндпоинта</summary>

| Параметр | Значение |
|----------|----------|
| Метод | `POST /Account/v1/User` |
| Content-Type | `application/json` |
| Обязательные поля | `firstName`, `lastName`, `userName`, `password`, `captchaToken` |

**Коды ответов:**

| HTTP | Код ошибки | Описание |
|------|------------|----------|
| 201 | — | Пользователь успешно зарегистрирован |
| 400 | 1100 | Отсутствуют обязательные поля |
| 400 | 1200 | CAPTCHA не пройдена |
| 400 | 1300 | Пароль не соответствует требованиям |
| 409 | 1204 | Пользователь уже существует |
| 500 | 1500 | Внутренняя ошибка сервера |

</details>
